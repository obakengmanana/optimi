
=========================================================================

Open/Load "Optimi" folderr in your code editor (Visual studio was used in this case)

On the terminal navigate to the project folder (Optimi)

Run command "npm install", this should install all the node modules 

Run command "npm install http-server" , this will alllow us to transfer data over the Hyper Text Transfer Protocol (HTTP)

Run command "http-server" to initiate the project.

After running this command, you will be provided with a url/ip address to use to accesss the project (e.g http://127.0.0.1:8080)



========================================================

On another terminal navigate to the project folder (Optimi) again

Run command "node server/service/index.js" this will initiate the server side


==============================================================

This project can be tested using requirements mentioned on the project specification document.
